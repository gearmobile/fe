# Fe

Современный адаптивный одностраничный сайт. Отлично подойдет для агенств или индивидуального использования.

# Плагины

При создании использовались следующие **плагины** и **ресурсы**:

* [Icomoon][1]
* [Google Maps][2]
* [Isotope][3]
* [Swiper Slider][4]
* [imagesLoaded][5]
* [Chartist.js][6]
* [Lost Grid][7]

# Верстка

Верстка производилась с использованием **шаблона** [TARS][8] из исходников в формате PSD ( **Adobe Photoshop** и [Adobe Creative Cloud][9] ).

## Кроссбраузерность

Поддержка браузерами:

* Firefox+
* Chrome+
* Opera+
* Safari+
* Internet Explorer 10+
